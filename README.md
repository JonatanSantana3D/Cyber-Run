## 🇧🇷/🇵🇹 Sobre Mim & O Projeto
Olá! 👋 Eu sou um **Artista 3D** apaixonado por criar mundos virtuais, e agora estou a dar os meus primeiros passos na programação de jogos! 

Este projeto foi desenvolvido enquanto faço o incrível curso do **Professor Henrique** na plataforma [Solo Game Dev](https://sologamedev.com.br/comece/). Graças a este curso, estou finalmente a conseguir trazer à vida os jogos que sempre sonhei criar. Misturar a minha arte 3D com a lógica de programação tem sido uma jornada fantástica!

🎮 **O que é o Cyber Run Ultra?**
Um jogo web super dinâmico com dois modos de jogo (Corrida e Espaço), onde o objetivo é desviar de obstáculos, recolher moedas, usar power-ups (Escudo 🛡️, Slow-Mo ⏳ e Ímã 🧲) e derrotar Chefões épicos para bateres o teu recorde!

---

## 🇺🇸/🇬🇧 About Me & The Project
Hi there! 👋 I'm a **3D Artist** with a passion for building virtual worlds, and now I'm taking my first steps into game programming!

This project was developed while taking the amazing course by **Professor Henrique** at [Solo Game Dev](https://sologamedev.com.br/comece/). Thanks to this course, I am finally bringing the games I’ve always dreamed of making to life. Mixing my 3D art skills with coding logic has been an incredible journey!

🎮 **What is Cyber Run Ultra?**
A highly dynamic web game featuring two game modes (Run and Space), where the goal is to dodge obstacles, collect coins, use power-ups (Shield 🛡️, Slow-Mo ⏳, and Magnet 🧲), and defeat epic Bosses to beat your high score!

---

### ✨ Features / Destaques
* 🏃‍♂️ **Modo Corrida (Run Mode):** Plataforma com saltos duplos / *Platformer with double jumps*
* 🚀 **Modo Espaço (Space Mode):** Sobrevivência com movimento lateral / *Survival with lateral movement*
* 👾 **Chefões Infinitos (Infinite Bosses):** Batalhas épicas a cada 1000 pontos / *Epic battles every 1000 points*
* 🛒 **Loja (Shop):** Desbloqueia novas cores com as tuas moedas / *Unlock new colors with coins*
* 🎵 **Música & SFX:** Banda sonora retro com controlo de volume / *Retro soundtrack with volume control*
